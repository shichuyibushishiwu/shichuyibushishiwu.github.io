# 金枪鱼插件

## 关于

金枪鱼是一个基于金枪鱼 `Revit API` 扩展包开发的 `Revit` 二次开发项目，也是基于 `Mvvmlight` 所尝试寻找的最佳实践，同时还是一个开发 `Revit` 样板的好助手。

当前版本支持 `Revit 2018.3-2024`。
